Oldschool
UI Style Guide
by Olivia Peterson

The goal of this UI Style Guide is to ease the documentation and the usage of the Oldschool brand guidelines in the digital world. Designers, front end developers and interactive designers can use this style guide to work more efficiently and keep brand consistency over all medias.

The Colours

The Oldschool colour palette consists of two natural, yet contrasting primary colours and two secondary colours that are bright and quirky to add pops of colour through the Oldschool brand. These colours will be the only colours used throughout the Oldschool branding and medias.

Typography

The Oldschool typography consists of one serif font, as Oldschools primary font and is used for headings and decorative design elements. Oldschools secondary font is a sans serif font and is used for blocks of text and information that needs to be easily and quickly read.

Buttons

The buttons used throughout the Oldschool branding and media channels are designed to be optimally designed for the user in mind. They are specifically designed to be aesthetically pleasing and interactive but to be easy to use and see.

Oldschool Logos

The Oldschool logo comes in two forms. The first is the Oldschool logo and this should be used where possible and only switched out for the second Oldschool brandmark logo when the logo needs to be used in small areas and the characters of the original Oldschool logo become unrecognisable.
The Oldschool logo minimum size is 300px x 83px and if smaller than that it should be switched out for the Oldschool brandmark logo.
